# DummyDB Documentation

**DummyDB** is a synthetic data generation pipeline that parses database schemas and produces realistic mock data using machine learning models (SDV).

## Supported Databases

| Type | Source | How |
|------|--------|-----|
| **SQL** | `.sql` file upload | Upload a `CREATE TABLE` dump |
| **PostgreSQL** | Supabase | Provide a Supabase connection string |
| **NoSQL** | MongoDB | Connect via `mongodb://` URI |
| **Graph** | Neo4j | Connect via `bolt://localhost:7687` |
